# Timetable #

Расписание на неделю при помощи LaTeX и TikZ.

### Использование ###

```bash
make          # создает pdf

make clean    # очищает временные файлы (кроме финальных)
make purge    # удаляет все, кроме исходников

make example  # собирает пример
```
